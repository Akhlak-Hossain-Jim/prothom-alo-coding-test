# ProthomAlo Coding test

Coding test for React Developer by ProthomAlo Bangladesh's industry-leading newspaper company.

## Overview

Created a basic Home & News details page form the design provided by the company as png file, with various functionality and approach including Atomic Designing model.

### `npm i`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.
